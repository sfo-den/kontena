# Users

## Register a New Kontena Account

```
$ kontena register
```

## Invite User to Kontena Master

```
$ kontena invite <email>
```

## Add User to Grid

```
$ kontena grid add-user <email>
```

## Remove User from Grid

```
$ kontena grid remove-user <email>
```
